# Features
This mod manager will help you to combine mods for Naruto Storm Connections.
- Character Mods (replace and add)
- Stage Mods (replace and add)
- Costume Mods (replace and add)
- Team Ultimate Jutsu Mods
- Resource Mods (for costume/ui mods)

# How to use mod manager?
First of all you have to select root folder for your game in option menu:
<img width="1306" height="798" alt="image" src="https://github.com/user-attachments/assets/4714d3e3-3073-47e8-9150-15520e73d1d1" />

After selecting root folder, you can install any mod with .nsc, .ensc. .uns, .unse format. That's formats for Mod Manager.
Mod manager also support old format .nus4, but some mods might work incorrectly.

To switch between games, press on Mod Manager Logo.

<img width="606"  alt="image" src="https://github.com/user-attachments/assets/24edffb2-a694-4947-9f1d-89d1441adfef" />



# How to install mod in mod manager?
It's pretty simple, just press on "Install Mod" button and select any mod with .nsc, .ensc. .uns or .unse format.
> [!IMPORTANT]
> Don't forget to select root folder of game!

<img width="1306" height="798" alt="image" src="https://github.com/user-attachments/assets/366a4d9f-de22-4fd5-b4e6-5411befe9746" />


# How to run mods with mod manager?
After installing mods, all you have to do is press "Compile Mods And Launch Game" button. Make sure you don't have data_win32 folder in root folder of game.
Once Mod Manager finish compilation, it will start game automatically.
> [!CAUTION]
> That process cleaning whole game from mods. Make sure you saved all mods which are stored in "ModdingAPI" folder somewhere else. For proper work, make sure you don't have installed mods in "data_win32" folder.
